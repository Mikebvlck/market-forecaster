-   **Neural Matrix ML Engine**: Active indicator consensus using Structural Trend, Momentum, and Liquidity Volume.
-   **Sniper Entry Logic**: Optimized entry points using pullback analysis for superior risk/reward ratios.
-   **Timeframe-Aware Strategy**: Dynamic volatility scaling for Scalping (1m) up to Macro Swing (Daily) trading.

## 📱 Global Deployment (How to Run on Phone/Other Computers)

To access your Neural Terminal from anywhere in the world, the best way is to host it on **GitHub Pages**.

### 1. Create a GitHub Repository
-   Go to **[GitHub.com](https://github.com)** and create a NEW repository called `market-forecaster`.
-   Keep it **Public** (so you can access the link on your phone).

### 2. Push Your Code (Run these on your Mac terminal)
Open your terminal in the project folder and run:
```bash
git add .
git commit -m "Initial launch of Kartoffel AI v8.0"
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/market-forecaster.git
git push -u origin main
```
*(Replace `YOUR_GITHUB_USERNAME` with your real username)*

### 3. Activate GitHub Pages
-   In your GitHub repository, go to **Settings > Pages**.
-   Under **Branch**, select `main` and `/ (root)`. Click **Save**.
-   GitHub will give you a link like: `https://YOUR_USER.github.io/market-forecaster/`

**Open that link on your iPhone, Android, or Tablet!** 🚀
